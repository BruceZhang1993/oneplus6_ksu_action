# 项目说明

使用 GitHub Actions 编译 OnePlus6 SukiSU Ultra 内核，基于 LineageOS 内核源代码

基于 LineageOS 22.2 OnePlus6 内核源码编译

## 支持特性  

### KernelSU

-[x] KernelSU 0.9.5 版本

### SukiSU Ultra

-[x] SukiSU Ultra NON-GKI 最新
-[x] SuSFS 1.5.5 (kernel-4.9) 开启全部特性支持
-[] KPM 支持

### 杂项

-[x] 移除内核版本号 -dirty 后缀
-[x] 增加应用隐藏补丁
-[] 增加 zram 算法补丁
